Servidor FTP

Um Servidor FTP é um sistema que permite a transferência de arquivos entre computadores por meio do protocolo FTP (File Transfer Protocol). Ele é muito usado para enviar, receber e gerenciar arquivos em servidores remotos, especialmente em hospedagens de sites.

JavaScript

O JavaScript (JS) é uma linguagem de programação usada principalmente para criar interatividade e dinamismo em páginas web. Ele roda no navegador do usuário e permite manipular elementos HTML e CSS em tempo real.

Docker

O Docker é uma plataforma que permite criar, empacotar e rodar aplicações dentro de containers. Um container é uma unidade leve e portátil que inclui o código da aplicação, suas dependências e bibliotecas necessárias para rodar em qualquer ambiente.
